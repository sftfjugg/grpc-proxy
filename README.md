# grpc-proxy
> The grpc dynamic proxy is based on rust.
> Provides non-invasive protocol conversion services.


## functional support

- [x] http to grpc
- [x] grpc annotations restful
- [x] custom metadata
- [ ] sidecar scheme
- [ ] dynamic reflect
- [ ] development(for event)

## quick start
```shell
cd example/restful

./example.sh
```
